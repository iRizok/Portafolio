## Implementacion de la Api

Utilizamos un Api que presta la NASA para poder visualizar en base a las fechas que utilicemos, una imagen tomada por la nasa de alguna galaxia o nebulosa, esta api lo que nos da como informacion es una descripcion breve de la foto, su ubicacion y fecha en la que fue tomada y tambien da un autor, dependiendo si el autor es publico.
